What is the result of executing the following code?

val sb = new StringBuilder("Hello, kitteh!")
val width = 20
1 to width - sb.length foreach sb.append(' ')
println(s"[$sb]")
  
  1. [Hello, kitteh!       ]

  2. [       ]

  3. [Hello, kitteh! ]

  4. [       kitteh!]
  5. Throws an exception, such as StackOverflowError or StringIndexOutOfBoundsException.

Explanation

The foreach takes a function with an Int parameter. Since StringBuilder.append returns a StringBuilder which is a function, that function is eta-expanded. The append is evaluated once at eta-expansion; each iteration invokes StringBuilder.apply, which is an alias for StringBuilder.charAt. If the StringBuilder had not been initialized with data, that invocation would throw StringIndexOutOfBoundsException.
